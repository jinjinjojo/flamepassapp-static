Mobile Not Supported

These games can only be downloaded and played on computers.

Mobile devices don't support running most of these games directly.

Please use a Windows, Mac, or Linux computer to download and play these games.

How to Play Offline

Follow these steps to play your downloaded game on a school computer:

1 Download the game to your personal computer
2 Extract the ZIP file using a program like WinRAR or 7-Zip
3 Copy all files to a USB drive (8GB+ recommended)
4 Plug the USB drive into your school computer
5 Find the game's main executable (usually a .exe file) and run it

Note: Some school computers may block running executables. In those cases, you might need to try one of our browser-based games instead.
